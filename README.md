# Clone YouTube

Este projeto é um clone da interface de reprodução de vídeo do YouTube, desenvolvido com o objetivo de praticar HTML5 semântico e CSS3 moderno.

## 🚀 Melhorias Recentes

O código foi completamente reescrito para garantir maior fidelidade ao design atual do YouTube (Modo Escuro), além de focar em **código limpo, organizado e totalmente comentado**.

### O que há de novo?

*   **Fidelidade Visual:** Cores, espaçamentos, tipografia e ícones ajustados para refletir a interface real do YouTube.
*   **HTML Semântico:** Uso de tags como `<nav>`, `<main>`, `<section>`, `<aside>` e `<article>` para melhor acessibilidade e SEO.
*   **CSS Moderno:** Utilização de Flexbox e CSS Grid para layouts complexos, além de variáveis CSS (`:root`) para facilitar a manutenção de temas.
*   **Responsividade:** O layout se adapta perfeitamente a diferentes tamanhos de tela (desktop, tablet e mobile).
*   **Interatividade Básica:** Adicionado um pequeno script JavaScript para demonstrar o funcionamento de botões como Like/Dislike, Inscrever-se e expandir a descrição.
*   **Acessibilidade:** Inclusão de atributos `aria-label`, `role` e navegação amigável.
*   **Código Limpo:** Todo o código (HTML, CSS e JS) está amplamente comentado, explicando a função de cada classe e estrutura.

## 🛠️ Tecnologias Utilizadas

*   **HTML5**
*   **CSS3** (Flexbox, Grid, Variáveis CSS)
*   **JavaScript** (Interatividade Vanilla)
*   **Google Fonts** (Fonte Roboto)
*   **Google Material Symbols** (Ícones da interface)

## 📂 Estrutura do Projeto

*   `youtube.html`: Estrutura principal da página.
*   `youtube.css`: Estilização completa do projeto.
*   `README.md`: Documentação do projeto.

## 💡 Como executar

1.  Clone este repositório em sua máquina local.
2.  Abra o arquivo `youtube.html` em qualquer navegador web moderno.
3.  Não é necessário nenhum servidor ou instalação de dependências.
